# Issues

Planned work items live here as one markdown file per issue.

## File naming
`NNN-short-title.md` — e.g. `001-game-board.md`

## Template

```markdown
# Title

## User story
As a … I want … so that …

## Acceptance criteria
- [ ] …

## Test expectations
- …

## Out of scope
- …

## Suggested owner
worker
```

# Architecture Decision Records

Document meaningful architecture decisions here.

## File naming
`NNN-short-title.md` — e.g. `001-state-management.md`

## Template

```markdown
# Title

## Status
Proposed | Accepted | Deprecated

## Context
What problem or constraint led to this decision?

## Decision
What was decided?

## Consequences
What trade-offs or follow-up work does this create?
```
